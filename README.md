## Installation

1. Clone the repo

   ```sh
   git clone https://github.com/Ekansh38/Pong.git

   ```

2. Install Dependencies

   ```sh
   pip install pygame

   pip3 install pygame

   python3 -m pip install pygame
   ```

3. Run the project
   ```sh
   python3 main.py
   ```
